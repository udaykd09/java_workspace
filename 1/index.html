MPTCP

This document describes different aspects of MultiPath Transmission Control Protocol (MPTCP). It is a protocol which attempts to use multiple Internet Protocol (IP) addresses at the same time. This means the protocol enables a Transmission Control Protocol (TCP) interface between applications, while using multiple subflows during the same connection. This paper describes the history, the current progress, and the next steps of MPTCP which are needed to make it a common, widely used protocol.
